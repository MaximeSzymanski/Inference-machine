Detailed Description

Fichier source des fonctions relative au craft.

Version
1.0
Date
25/12/2020
Author
Maxime SZYMANSKI Jules PETRY
  1. Lecture des regles dans un fichier : Les regles sont lues dans un fichier afin de construire la base de connaissance. Les regles sont les items que le joueur peut craft avec les ressources necessaires.
  2. Moteur d'inference :
    Le moteur d'inference permet de donner les crafts possible en fonction de l'inventaire du joueur.
