Detailed Description

Fichier source des fonctions relative a l'arbre et donc la map du jeu.

Version
1.0
Date
25/12/2020
Author
Maxime SZYMANSKI Jules PETRY
  1. Creation de l'arbre : En premier lieu, un arbre est cree avec une racine et deux feuilles au tout debut du jeu.
  2. Gestion de l'arbre : \nDes feuilles peuvent etre ajoute en fonction de l'evolution du joueur // l'arbre est detruit a la fin du jeu.
